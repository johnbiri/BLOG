Diário de Estudos - #001

Hoje comecei minha jornada no mundo da criação de jogos com Unity, motivado pela vontade de dar vida a ideias únicas e explorar minha criatividade. Criar um diário como este é importante para registrar meu progresso, entender o que já aprendi e manter a disciplina nos estudos. Neste primeiro dia, instalei a versão gratuita do Unity, explorei o ambiente e já dei alguns passos concretos. Aprendi a criar objetos básicos como cubos e terrenos, apliquei texturas para personalizar os elementos, e implementei movimentação simples usando o Vector3. Também comecei a explorar as ferramentas de animação, como o Animation e o Animator, que são essenciais para trazer vida aos personagens. Além disso, descobri sites úteis para baixar assets como modelos de personagens, texturas e animações que facilitarão meu trabalho. Estou animado para continuar e ver onde essa jornada vai me levar!

15 de dezembro de 2024


Eu criei e publiquei um site estático no GitHub Pages para aplicar meus conhecimentos em HTML, CSS, Bootstrap e GitHub. Primeiro, desenvolvi um site básico com uma página principal (index.html) usando meu editor de código preferido, o Visual Studio Code (VSCode). Após finalizar o site, criei um novo repositório no GitHub e o nomeei de acordo com o propósito do meu site, por exemplo, “BLOG”. Em seguida, adicionei o repositório remoto ao meu projeto com o comando git remote add origin [URL do meu repositório] e fiz o upload dos arquivos com git push origin main. Verifiquei se os arquivos foram corretamente enviados para o GitHub e ativei o GitHub Pages em Settings > Pages, selecionando a branch “main” como a fonte e clicando em “Save”. Após alguns minutos, copiei o URL fornecido pelo GitHub e testei o site em meu navegador para garantir que tudo estava funcionando como esperado. Por fim, compartilhei o URL do meu site com meus colegas de classe. Esta atividade me proporcionou uma prática valiosa na criação e publicação de sites, solidificando meus conhecimentos em desenvolvimento web e me familiarizando com ferramentas essenciais do desenvolvimento moderno.
